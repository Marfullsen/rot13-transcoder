## Desc

Site for encrypting and decrypting ciphertext with rot13.

---

## Usage localhost
1. Install `git`.
2. Clone, enter the directory and open directly.
``` 
git clone https://github.com/Marfullsen/rot13-transcoder.git
cd rot13-transcoder
```
3. Have fun!
